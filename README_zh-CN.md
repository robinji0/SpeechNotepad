# 🎙️ 精准语音输入 (Speech Notepad)

[🇨🇳 简体中文](README_zh-CN.md) | [🇬🇧 English](README.md)

**精准语音输入 (Speech Notepad)** 是一款主打“断点接续、指哪打哪”的智能 Chrome 语音转文字插件。

它彻底解决了市面上绝大多数语音插件的核心痛点：**粗暴覆盖**。使用本插件，你可以随时中断录音、手动修改错别字、然后将光标放在任意位置继续说话——新识别的文字会**完美插入到光标所在处**，绝对不会覆盖或弄乱你之前的心血。

## ✨ 核心亮点

* 🎯 **断点接续，精准插入：** 光标在哪里，文字就打在哪里。完美支持“录音 -> 暂停手动修改 -> 继续录音”的无缝工作流。
* ⌨️ **全局快捷键 (跨软件唤起)：** 谁说 Chrome 插件只能在浏览器里用？配置全局快捷键（如 `Ctrl+Shift+S`），即使你在用 IDEA 写代码、用 Word 写文档，也能一键呼出悬浮语音记事本！
* 🌍 **独立语种识别：** 界面语言与识别语言彻底解耦。原生支持中、英、日、韩等 7 种语言的精准识别，完美解决中英夹杂或系统语言导致的“空耳”乱码问题。
* 📋 **一键复制与流转：** 点击“复制全部”，自动为你停止录音、复制文本并清空输入框，行云流水，为下一次灵感腾出空间。
* 🕒 **本地历史记录：** 自动暂存你最近的转换记录。不小心清空了？手滑关掉了？点击历史记录即可一键恢复。
* 🌗 **双语界面：** 支持中英双语 UI 实时切换。

## 🚀 安装指南

目前可以通过 Chrome 开发者模式快速加载使用：

1. 下载或 `git clone` 本仓库代码到你的电脑上。
2. 打开 Chrome 浏览器，在地址栏输入并前往 `chrome://extensions/`。
3. 打开页面右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”** 按钮。
5. 选择你刚才存放代码的文件夹。
6. 在浏览器工具栏将插件“固定”显示，方便随时点击！

## 💡 如何配置跨软件全局快捷键？

想在写代码 (IDEA/VSCode) 或聊微信时随时随地语音输入？

1. 在 Chrome 地址栏输入 `chrome://extensions/shortcuts` 进入快捷键设置页。
2. 找到 **Speech Notepad** 插件区域。
3. 在“全局唤起语音记事本 (跨软件)”一栏，录入你习惯的快捷键（例如 `Ctrl+Shift+S` 或 Mac 上的 `Cmd+Shift+S`）。
4. **最关键的一步：** 将右侧的下拉框从“在 Chrome 中”修改为 **“全局 (Global)”**。
5. 把 Chrome 最小化，打开任意其他软件，按下快捷键，尽情体验吧！

## 🔒 隐私声明

本插件底层调用 Chrome 原生 Web Speech API，所有语音识别过程均由浏览器直接处理。我们**不包含任何后端服务器**，**绝不**收集、存储或上传您的任何语音数据和文本。您的转换历史仅通过 `localStorage` 安全地保存在您的本地设备上。
